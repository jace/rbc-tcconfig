Configuration Page

When deploying a new machine, update these configuration settings in the listed sequence.

  1. Configure IP settings and machine host name.
  2. Reboot. This is required to make the host name change apply.
  3. Dial up to the internet (Tata Indicom locations only).
  4. Check that the internet connection is working.
  5. Create a new host key for the machine.
  6. Authorise the computer with the server.
  7. Configure the printer.
  8. Install all available software updates.
